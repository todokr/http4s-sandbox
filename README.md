# Http4s Sandbox

## API Server

- Http4s
- Smithy4s 

## Web Client

- React + TypeScript
- Vite + Vitest + SWC
- Biome

## Directory Structure
```
.
├── server (Http4s)
│   └── src/main/smithy (API IDL)
├── api-client (Generated by Smithy4s)
└── web (React)
```

## How to

### Run API Server

```shell
cd server
sbt run
```

### Run Web Client

```shell
cd web
pnpm i
pnpm dev
```

### Generate API Interface (Server)

```shell
cd server
sbt "smithy4sCodegen"
```

### Generate API Client (Web)

requires [smith-cli](https://smithy.io/2.0/guides/smithy-cli/index.html)

```shell
cd api-client
smithy build
```
